# User Stories

Backlog of user-story-shaped features. Promoted into Issues when ready.

## MVP stories
- US-01 As a visitor, I want to see what MonadMon is in < 30 seconds, so I decide to connect.
- US-02 As a visitor, I want to connect my wallet on Monad testnet, so I can start.
- US-03 As a new player, I want to mint one Genesis Egg, so I have a creature to raise.
- US-04 As a new player, I want to hatch my egg and see what I got, so I feel ownership.
- US-05 As a player, I want to see my Monster's stats, DNA, and type clearly, so I understand it.
- US-06 As a player, I want to train my Monster on a cooldown, so I can grow it.
- US-07 As a player, I want to challenge another player to a battle, so I can prove my Monster.
- US-08 As a player, I want to see who wins and why, so the outcome feels fair.
- US-09 As a player, I want to climb a leaderboard, so I have a goal.
- US-10 As a spectator, I want to see another player's showcase, so I can compare.

## Phase 2+ stories (deferred)
- US-11 As a player, I want to trade Monsters on a marketplace.
- US-12 As a player, I want to use items (stones, candies) to evolve my Monster.
- US-13 As a player, I want to explore regions and find rare items.
- US-14 As a guild leader, I want to organize tournaments.
- US-15 As a player, I want to fuse two Monsters into a new one.

Each story gets promoted to a Feature Issue with full AC when picked.
